Instruction for compiling the program:
	-> The program is compiled using a makefile which is attached.
	-> Type 'make' in the terminal to compile the program file(s).

Instruction for running the program:

	-> Type in the terminal :- './Search'
	-> Examples: 

		./Search /home/downloads

		./Search -f txt

		./Search -s 1000

		./Seach -s 1000 -e "ls -l"

		./Seach -s 1000 -e "wc"

		./Seach -s 1000 -E "ls -l"
